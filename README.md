# 🎬 CinePrompt Studio v2.0

Screenplay to shot-by-shot AI image prompts generator.

## Deploy to Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel` in this folder
3. Follow prompts — done!

## Or Deploy via GitHub

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import repo
3. Auto-deploys!

## Features

- 12 AI providers (Gemini, Claude, GPT, DeepSeek, Groq, etc.)
- Character sheets with description prompts
- AI auto-decides camera, lighting, style
- Screenplay parser (INT./EXT. format)
- Storyboard grid & timeline views
- Export: TXT, JSON, CSV, Markdown
- Data persists via localStorage
